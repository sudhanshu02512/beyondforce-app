Apex Classes

Apex class is a template or blueprint to create objects.

Frontend  - HTML is a template

Backend - Class is a template

What is an object?

In real life an object is a thing that you can see and touch and when you touch then that thing can also behave based on their behaviour.

In technical terms objects are things that have some properties(What the Object HAS) and have behaviour(give some output on interaction or what the object DOES).

Other Examples

  1. Person : Person has name, age, gender and can talk, walk and sleep.
  1. Properties: Name, Age, Gender, etc.
  2. Behaviour: Walk, Talk, Sleep, etc.

In apex an object is something that is created using an apex class.

Apex classes have two things.

  1. Apex Class Variables(Properties or Data) : Variables are used to store data or in other words used to store properties of an object.
  2. Apex Class Methods(Behaviour or action) : Methods are used to define the behaviour of objects.
