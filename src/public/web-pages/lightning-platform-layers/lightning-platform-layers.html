Salesforce Lightning Platform Layers

Presentation Layer

UI where users interact

Components : LWC, Aura, Lightning Experience, Mobile

Application Logic Layer

Handles business rules & processes

Components: Apex, Triggers, Flows, Validation Rules

Integration Layer

Connects Salesforce with external systems

Components: REST, SOAP, Bulk API, Platform Events

Security Layer

Controls access and permissions

Components: Auth, Profiles, Permission Sets, Sharing, FLS, CRUD

Persistence Layer

Stores and manages data

Components : Objects, Records, SOQL, DML, Transactions

Metadata Layer

Defines configuration & behavior

Components: Object definitions, Layouts, Flows, Custom Labels

Platform / Infrastructure Layer

Underlying cloud infrastructure (Salesforce-managed)

Components Servers, DB, scaling, backups
